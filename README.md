# 🌟 Tarot Cards - 神秘塔罗占卜助手 🌟

一个专注于女性用户体验的iOS塔罗牌占卜应用，融合了传统智慧与现代技术，为用户提供精准而温暖的运势指引。

## 💕 关于我们

这是一个承载着两个人智慧与情感的项目：
- **老萨满** - 技术架构师，负责将创意转化为可执行的代码
- **陈柔** - 产品设计师 & 经验丰富的占卜指导师，专注于用户体验与情感连接的细节打磨

作为一名教师，我深信技术应该服务于人的成长和幸福。这个项目不仅是技术的实践，更是我教学理念的延伸——让复杂的知识变得简单易懂，让神秘的文化变得亲切可感。

## ✨ 核心功能

### 🎯 三张牌占卜 - 像上课一样清晰的学习体验
- 随机抽取三张塔罗牌，包含正位逆位详细解读
- 精美的翻牌动画效果，提升占卜仪式感
- 完整的历史记录功能，回顾之前的占卜结果
- 支持自定义问题记录，让每一次占卜都有针对性

*作为你的占卜导师，我会引导你理解每一张牌的深层含义，就像在课堂上耐心讲解每一个知识点一样。*

### 💖 精美分享功能 - 分享知识的快乐
- 一键生成高质量分享卡片，包含完整的占卜信息
- 采用粉紫渐变配色方案，符合女性审美偏好
- 支持真实卡牌图片展示，提升分享的专业度
- @2x高清画质，确保在各种平台上显示效果优秀

*分享是学习过程中最重要的环节之一。通过分享，你不仅能巩固自己的理解，还能帮助更多人了解塔罗文化的魅力。*

### 🌅 每日一签 (V1.3) - 每天的智慧小课堂
- 智能单卡抽取，每日运势精准指引
- 时间感知问候系统，根据时段提供个性化体验
- 连续签到奖励机制，培养用户的占卜习惯
- 30天历史回顾功能，见证运势变化轨迹
- AI智能解析引擎，基于GLM模型的个性化分析

*就像每天都要给学生布置适当的作业一样，我也希望你养成每天占卜的习惯。这不是迷信，而是对自己内心状态的觉知和反思。*

## 🛠 技术栈 - 现代化的教学工具

- **开发语言**: Swift - 优雅而现代的编程语言
- **UI框架**: UIKit - 成熟稳定的界面框架
- **布局工具**: SnapKit - 灵活的约束布局系统
- **数据处理**: SwiftyJSON - 简洁的数据处理方式
- **依赖管理**: CocoaPods - 高效的项目管理工具
- **AI模型**: GLM-4.7-Flash - 先进的人工智能技术
- **版本控制**: Git - 协同开发的标准工具

## 🚀 快速开始 - 简单易懂的安装指南

```bash
# 克隆项目 - 就像复制课堂笔记一样简单
git clone https://github.com/alucardulad/Tarot-Cards.git
cd Tarot-Cards

# 安装依赖 - 准备学习工具
pod install

# 打开项目 - 开始我们的课程
open tarot_cards.xcworkspace
```
