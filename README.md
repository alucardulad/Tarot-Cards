# 🌟 Tarot Cards - 神秘塔罗牌占卜助手 🌟

一个专注于女性用户体验的iOS塔罗牌占卜应用，融合了传统智慧与现代技术，为用户提供精准而温暖的运势指引。

## ✨ 核心功能

### 🎯 三张牌占卜 - 像上课一样清晰的占卜体验

- 随机抽取三张塔罗牌，包含正位逆位详细解读
- 精美的翻牌动画效果，提升占卜仪式感
- 完整的历史记录功能，回顾之前的占卜结果
- 支持自定义问题记录，让每一次占卜都有针对性

*作为你的占卜导师，我会引导你理解每一张牌的深层含义，就像在课堂上耐心讲解每一个知识点一样。*

### 🌌 鉴赏模式 - 24张塔罗牌星空画廊

- 网格展示全部24张塔罗牌，简洁优雅
- **星空粒子特效**：多种星星混合（白/淡白/浅紫/深紫）
- **漂浮光球**：紫色 + 青色光晕，缓缓飘动
- **流星效果**：偶尔划过，增添神秘感
- **宇宙尘埃**：深紫色微点，营造层次感
- **呼吸光晕**：卡片边缘柔和发光，呼吸效果
- **浮动动画**：卡片轻微上下浮动，像云在飘
- 点击卡片进入详情页，查看完整含义

*沉浸式星空氛围，让你在欣赏塔罗牌的同时，感受到宇宙的神秘与宁静。*

### 🃏 卡牌详情页 - 精美大图 + 完整解读

- 卡牌大图展示，紫色光晕呼吸效果
- **标题渐变**：紫色到青色的渐变效果
- 正位/逆位含义完整展示
- 标签清晰，易于阅读
- 返回按钮、填充信息按钮

*每一张塔罗牌都有它的故事，在这里，你可以慢慢品味它的深意。*

### 💖 精美分享功能 - 分享知识的快乐

- 一键生成高质量分享卡片，包含完整的占卜信息
- 采用粉紫渐变配色方案，符合女性审美偏好
- 支持真实卡牌图片展示，提升分享的专业度
- @2x高清画质，确保在各种平台上显示效果优秀

*分享是学习过程中最重要的环节之一。通过分享，你不仅能巩固自己的理解，还能帮助更多人了解塔罗文化的魅力。*

### 🌅 每日一签 (V1.3) - 每天的智慧小课堂

- 智能单卡抽取，每日运势精准指引
- 时间感知问候系统，根据时段提供个性化体验
- 连续签到奖励机制，培养用户的占卜习惯
- 30天历史回顾功能，见证运势变化轨迹
- AI智能解析引擎，基于GLM模型的个性化分析

*就像每天都要给学生布置适当的作业一样，我也希望你养成每天占卜的习惯。这不是迷信，而是对自己内心状态的觉知和反思。*

### 🎨 粒子系统 - 可复用的粒子特效库

作为工具类封装，提供简单API：

```swift
// 添加星空粒子
ParticleManager.addStarfield(to: view)

// 添加光球粒子
ParticleManager.addOrbs(to: view)

// 添加流星
ParticleManager.addShootingStars(to: view)

// 添加尘埃
ParticleManager.addDust(to: view)

// 添加完整效果（星星 + 光球 + 流星 + 尘埃）
ParticleManager.addFullEffects(to: view)

// 自动清理
deinit {
    ParticleManager.cleanup(for: view)
}
```

*清晰的API，简单的调用，强大的效果。*

## 🌌 视觉特色

### 紫色氛围系统

整个应用采用统一的紫色系配色：

- **深紫背景**: 2D1344 -> 1E1233 -> 120632
- **主色**: 7D3FE1（紫色）
- **辅助色**: A5F2FF（青紫色）
- **文字色**: F8F8FF（浅白）
- **背景色**: F5C518（琥珀色，用于强调）

### 粒子特效层次

1. **星空粒子**：大星星、中等星星、小星星、微小星星
2. **光球粒子**：紫色 + 青色光晕，缓慢漂浮
3. **流星粒子**：白色拖尾，快速划过
4. **尘埃粒子**：深紫色微点，营造背景质感

### 动画效果

- **卡片浮动**：轻微上下移动，像云在飘
- **呼吸光晕**：边缘发光，透明度呼吸变化
- **点击反馈**：缩放 + 光晕增强
- **翻牌动画**：从背面翻到正面

## 🚀 快速开始 - 简单易懂的安装指南

### 安装步骤

```bash
# 克隆项目
git clone https://github.com/alucardulad/Tarot-Cards.git
cd Tarot-Cards

# 安装依赖
pod install

# 打开项目
open tarot_cards.xcworkspace
```

### 编译运行

1. 在 Xcode 中选择目标设备（iOS 模拟器或真机）
2. 点击运行按钮（⌘R）
3. 开始你的塔罗占卜之旅

## 📚 使用指南

### 首页操作流程

1. **查看抽卡** - 页面显示三张卡牌背面
2. **点击"去抽卡"** - 进入抽卡页面
3. **等待翻牌** - 精美的翻牌动画
4. **查看结果** - 卡牌名称 + 含义解读
5. **点击"再次抽卡"** - 重新抽3张新牌
6. **历史记录** - 查看之前的占卜结果

### 鉴赏模式操作流程

1. **点击"✨ 鉴赏模式"** - 进入星空画廊
2. **浏览卡牌** - 滑动查看24张塔罗牌
3. **点击任意卡** - 进入详情页
4. **查看详情** - 大图 + 完整含义
5. **返回继续浏览** - 实现沉浸式鉴赏体验

### 每日一签操作流程

1. **选择问题** - 输入你的问题（可选）
2. **点击"去抽卡"** - 单卡抽取
3. **查看运势** - 每日签结果 + 解读
4. **签到打卡** - 连续签到奖励
5. **历史回顾** - 查看过去30天的运势
